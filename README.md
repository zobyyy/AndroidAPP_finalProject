# Android APP 大學期末專題成果

這是一個以寵物日常生活照護為主題的APP，其主要功能為**註冊**、**登入**、**CRUD**

在團隊中我負責基本**程式碼撰寫**及**介面排版**

使用語言：Java

開發環境：Android Studio
